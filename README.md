# Customer Shopping Behavior Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior using SQL and Power BI. The dataset was explored to identify customer purchasing patterns, product category performance, sales trends, and customer demographics.

The main objective of this project is to derive meaningful business insights that can help improve customer engagement, marketing strategies, and sales performance.

---

## Tools & Technologies Used
- PostgreSQL
- SQL
- Power BI
- Excel

---

## Project Workflow
1. Imported and cleaned the customer shopping dataset
2. Performed data analysis using SQL queries
3. Extracted key business insights from customer purchase behavior
4. Built an interactive Power BI dashboard for visualization
5. Generated insights and recommendations based on trends

---

## Key Business Questions Solved
- Which product category generated the highest sales?
- Which age group contributes the most to purchases?
- What are the purchasing trends among customers?
- Which customer segments are most active?
- How do customer demographics impact shopping behavior?

---

## Key Insights
- Electronics and Fashion categories generated high customer engagement
- Customers aged 25–40 were among the most active buyers
- Female customers showed higher purchasing frequency in several categories
- Seasonal shopping trends were visible across product categories
- Customer segmentation helped identify high-value customer groups


---

## Dashboard Features
The Power BI dashboard includes:
- Sales overview
- Product category analysis
- Customer demographic analysis
- Purchase trend visualization
- KPI cards and charts
- Interactive filters and slicers
